Para executar o programa, primeiramente faça o clone do projeto para sua maquina.

## Instalação

Clone

```bash
  git clone
```

Dentro da pasta do projeto instale as dependencias necessárias

```bash
  npm install
```

Rode o projeto

```bash
  npm run start
```
